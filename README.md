# BUDDIES Hub

BUDDIES Hub is a lightweight front-end starter app that showcases **full sectors** where teams can build products.

## Included

- Hero section introducing BUDDIES mission.
- Filterable sector cards (Core, Emerging, Impact).
- Basic roadmap section for build phases.

## Run locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
